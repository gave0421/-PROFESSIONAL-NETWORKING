Vibe Coding Hackathon

Howest

Tijdens de Vibe Coding Hackathon bij Howest moesten we op één dag een idee uitwerken tot een werkende demo. De focus lag niet op zelf programmeren maar op snel bouwen met AI-tools zoals Lovable.

Ons idee was een vluchtplanner voor studenten. Voor wie graag op reis wil maar nog niet weet waarheen: op basis van goedkope vluchten krijg je wat inspiratie om sneller iets te plannen.

Wat me opviel, is dat vibe coding vooral draait om goed uitteleggen wat je wil. Je typt minder code maar je geeft instructies en je test het en stuurt bij.

Eerlijk gezegd vond ik de hackathon zelf wat tegenvallen. We zaten met vier à vijf man in een groep maar konden maar met één prompt tegelijk werken. Daardoor zaten we vaak naar één scherm te kijken en veel te wachten. Dat werd op den duur saai.

Het resultaat viel wel mee. In korte tijd hadden we een mooie, duidelijke website. Voor een snelle demo werkt vibe coding dus prima maar voor echte software blijft goede, begrijpbare code belangrijk.
