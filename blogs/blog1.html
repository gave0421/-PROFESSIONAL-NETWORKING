De tech stack

Deze website is een statische site die ik gemaakt heb met gewoon HTML, CSS en een beetje JavaScript. HTML voor de structuur, CSS voor de opmaak en de responsive layout, en JavaScript voor de kleine dingen zoals de navigatie. Ik heb het bewust simpel gehouden. De site staat online via GitHub Pages, dat is gratis en zet je pagina’s meteen vanuit je repository online.

Gemaakt met GitHub Copilot

Ik heb de site “vibe coded” met GitHub Copilot. Ik gaf gewoon uitleg over wat ik wou en liet Copilot code genereren, Ik testte het en paste het daarna aan. Zo ging het een stuk sneller dan alles zelf uittypen.

Maar ik moest nog altijd snappen wat er stond. Copilot helpt maar de code moet wel kloppen en leesbaar blijven. Ik heb dus telkens goed nagekeken wat er gegenereerd werd en heb het bijgestuurd waar nodig was.
