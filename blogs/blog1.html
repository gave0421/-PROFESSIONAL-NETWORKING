Hoe deze website tot stand kwam

Gebouwd met HTML, CSS, JavaScript en GitHub Copilot

Deze website is geen CMS, maar een statische website die opgebouwd is met HTML voor de structuur, CSS voor de styling en JavaScript voor de kleine interacties. Alles is bewust overzichtelijk gehouden zodat de blogposts en de navigatie snel leesbaar blijven.

De structuur van de site bestaat uit een overzichtspagina met blogkaarten en aparte blogpagina’s per artikel. Daardoor kan elke post zijn eigen inhoud, titel en afbeelding hebben, terwijl de rest van de website dezelfde look and feel behoudt.

Voor de styling is CSS gebruikt om de pagina rustig, responsive en consistent te maken. De kaarten, knoppen en bloglayout zorgen samen voor een moderne maar simpele visuele stijl die op verschillende schermformaten goed werkt.

JavaScript zorgt voor de extra dynamiek op de website. De site bevat geen zware logica, maar wel genoeg interactie om de ervaring vlotter te maken en de inhoud beter te laten aanvoelen als een echte blog.

Deze website is ook vibe coded met GitHub Copilot. Dat betekent dat ik sneller kon itereren, ideeën kon testen en code kon bijsturen met hulp van AI. De code blijft wel belangrijk: de structuur, CSS en JavaScript moeten nog altijd leesbaar en logisch blijven.

De website werd online gezet met GitHub Pages. Wie een nieuwe post wil toevoegen, moet een nieuwe HTML-pagina aanmaken, de inhoud en afbeelding invullen, en daarna ook de homepage-card en navigatie bijwerken. Dat is meer werk dan in een CMS, maar het blijft wel heel duidelijk en controleerbaar.
